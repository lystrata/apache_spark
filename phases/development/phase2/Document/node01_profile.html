ROTA=0 on every device (non-rotational). MODEL=LOGICAL VOLUME means the HPE Smart Array is presenting these as RAID logical volumes — the Smart Array does not support JBOD/passthrough on this hardware, so per-disk single-drive RAID0 LVs is the path to meet Proxmox's JBOD requirement. The single shared serial PZDLA0B52KQ0TF is the controller's identifier, not a per-disk serial.
