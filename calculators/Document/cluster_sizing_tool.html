Revisions · Tool change history
2026-04-18 — Initial Release

General-purpose cluster sizing tool derived from the dev and production calculator math. All hardware parameters (nodes, cores, RAM, NVMe count, NVMe size, OSD count, RAID type) are exposed as inputs rather than baked-in constants, allowing any cluster configuration to be modelled. Key addition over the fixed calculators: the Shuffle Amplification Ceiling metric, which directly answers "how complex a query can this hardware sustain at the configured ingest volume" without requiring manual scratch calculation.
